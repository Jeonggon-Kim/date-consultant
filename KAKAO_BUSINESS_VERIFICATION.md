# 카카오 비즈니스 인증 가이드

## 📋 개요

카카오 로그인에서 **이메일 정보를 필수로 수집**하려면 **카카오 비즈니스 인증**이 필요합니다.

---

## 1. 비즈니스 인증이 필요한 이유

### 권한 제한 항목
다음 항목들은 비즈니스 인증 없이는 **"권한 없음"** 상태입니다:
- ❌ 카카오계정(이메일) - **필수 동의**
- ❌ 이름
- ❌ 성별
- ❌ 연령대
- ❌ 생일
- ❌ 출생 연도
- ❌ 전화번호

### 비인증 상태에서 사용 가능한 항목
- ✅ 닉네임 (선택 동의)
- ✅ 프로필 사진 (선택 동의)
- ✅ 카카오계정(이메일) - **선택 동의만 가능**

---

## 2. 비즈니스 인증 절차

### 2.1 사업자 정보 준비
다음 서류가 필요합니다:
- **사업자등록증** (개인사업자 또는 법인)
- **대표자 신분증**
- **통신판매업 신고증** (선택사항, 있으면 유리)

### 2.2 카카오 비즈니스 인증 신청

1. **카카오 개발자 콘솔** 접속
   - https://developers.kakao.com/

2. **내 애플리케이션** 선택

3. **비즈니스** 탭 클릭

4. **비즈니스 인증 신청** 클릭

5. 정보 입력:
   - **사업자 유형**: 개인사업자 / 법인사업자
   - **사업자등록번호**: 10자리 숫자
   - **상호명**: 사업자등록증의 상호명
   - **대표자명**: 사업자등록증의 대표자명
   - **사업장 주소**: 사업자등록증의 주소

6. **서류 업로드**:
   - 사업자등록증 사본 (PDF, JPG, PNG)
   - 대표자 신분증 사본 (선택)

7. **제출** 클릭

### 2.3 심사 기간
- **일반적으로 1~3 영업일** 소요
- 서류 보완 요청 시 추가 시간 필요
- 심사 결과는 이메일로 통보

---

## 3. 비즈니스 인증 완료 후 설정

### 3.1 동의 항목 활성화

비즈니스 인증이 완료되면:

1. **제품 설정** → **카카오 로그인** → **동의 항목**

2. **카카오계정(이메일)** 항목:
   - "설정" 클릭
   - **필수 동의** 또는 **선택 동의** 선택 가능
   - 저장

3. 다른 개인정보 항목도 활성화 가능:
   - 이름, 성별, 연령대, 생일 등

### 3.2 코드 활성화

`app/components/AuthModal.tsx` 파일에서 주석 해제:

```tsx
// 현재 (비활성화)
{/* 
<div className="relative my-6">
  ...카카오 로그인 버튼...
</div>
*/}

// 활성화 후
<div className="relative my-6">
  ...카카오 로그인 버튼...
</div>
```

주석(`{/* */}`)을 제거하면 카카오 로그인 버튼이 다시 표시됩니다.

---

## 4. 비즈니스 인증 없이 사용하는 방법

### 4.1 이메일을 선택 동의로 설정

비즈니스 인증 없이도 **선택 동의**로는 사용 가능:

1. **동의 항목** → **카카오계정(이메일)**
2. "설정" 클릭
3. **선택 동의** 선택
4. 저장

**장점:**
- 비즈니스 인증 불필요
- 사용자가 이메일 제공 선택 가능

**단점:**
- 사용자가 이메일 제공을 거부할 수 있음
- 이메일 없는 사용자 처리 필요

### 4.2 대안: 다른 소셜 로그인 사용

**구글 로그인** (권장):
- 비즈니스 인증 불필요
- 이메일 기본 제공
- 설정 간단

**네이버 로그인**:
- 비즈니스 인증 불필요
- 이메일 기본 제공

## 5. 문제 해결

### "비즈니스 인증이 거부되었습니다"

**원인:**
- 사업자등록증 정보 불일치
- 서류가 불명확
- 사업자 유형 불일치

**해결:**
1. 거부 사유 확인 (이메일 통보)
2. 서류 재확인
3. 재신청

### "이메일 항목이 여전히 권한 없음"

**원인:**
- 비즈니스 인증 미완료
- 캐시 문제

**해결:**
1. 비즈니스 인증 상태 확인
2. 브라우저 캐시 삭제
3. 카카오 개발자 콘솔 재로그인
